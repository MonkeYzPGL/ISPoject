Client 	-> vede datele tuturor clientilor
	-> adauga un client
	-> creaza un proiect
	-> adauga o cerere pentru proiect
	-> poate vedea datele unui client
	-> poate sterge un client

Angajat -> isi actualizeaza procentul pentru proiect
	-> adauga un nou angajat
	-> adauga cerere de marire
	-> poate vedea toti angajatii
	-> poate vedea un angajat
	-> poate vedea proiectul la care este asignat
	-> poate vedea toti angajatii dintr-o echipa

TeamLeader 	-> poate vedea datele tuturor teamleaderilor
		-> poate vedea datele unui teamleader
		-> poate vedea proiectul unui teamleader
		-> poate vedea echiipa unui teamleader
		-> actualizeaza procentul TeamLeaderului luat de la angajatii din echipa
		-> adauga un nou teamleader
		-> adauga un angajat in echipa
		-> adauga teamleader la echipa
		-> poate actualiza procentajul total al echipei (PUT)
		-> sterge un angajat din echipa
		-> sterge un teamleader

CEO 	-> vede datele tuturor CEO, angajati, teamleaderi, proiectelor, echipelor, clientilor, cereri de proiect, cereri de marire 
	-> vede datele unui/unei CEO, angajat, teamleader, proiect, echipe, client, cerere de proiect, cerere de marire
	-> poate adauga un CEO, teamleader, angajat, echipa
	-> poate schimba starea cererii de marire angajatului (sa fie aprobata sau nu)
	-> poate schimba salariul TeamLeaderului si al angajatului (fara a fi nevoie de cerere)
	-> promoveaza un angajat la TeamLeader
	-> poate aproba/refuza un proiect
	-> schimba salariul angajatului daca cererea sa de marire de salariu este aprobata
	-> poate asigna un proiect unei echipe
	-> poate asigna un angajat la echipa
	-> poate adauga un TeamLeader intr-o echipa	
	-> sterge CEO,  teamleder, echipa, angajat
	-> poate sterge liderul echipei
